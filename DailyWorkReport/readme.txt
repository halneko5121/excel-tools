1.概要
==================================
　作業日報の excel 生成を簡略化するためのスクリプトです。
　社員が増えた際の対応。その他、負担軽減が主な目的です。

2.スクリプトの利用方法
==================================
　1.　テンプレートとなる作業日報 excel を作成します。
　2.　「TemplateParam.xls」にてパラメータを設定して下さい。
　3.　「Template.bat」を実行して下さい。
　4.　「Users」フォルダに設定した社員分の日報 excel が出力されます。 

　何か不具合がありましたら、担当者にお知らせください。

3.利用方法詳細
==================================
　・テンプレートの　excel
　　「Tools/Template」フォルダの中に「Template.xls」があります。
　　この excel を元に社員分の日報を生成します。

　　※「日報」という名前のシートを用意して下さい。
　　　　それ以外のシート（書き方の説明とか）はいくらあっても問題ありません。
　　　　ただ、日報シートは順番的に最後にくるようにして下さい。

　・パラメータ
　　「Tools/Template」フォルダの中に「TemplateParam.xls」があります。
　　この中の各種パラメータを操作して、社員分の日報 excel を作成する事になります。

　　各種パラメータにつきましては、excel のコメント欄で説明していますのでそちらを参照下さい。

　・実行
　　パラメータが設定できましたら、「Template.bat」を実行して下さい。
　　トップの「Users」フォルダに人数分の 日報 excel が生成されます。
