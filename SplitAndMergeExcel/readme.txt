1.概要
==================================
　excel ファイルの分割・結合を行うスクリプトです
　

2.分割について
==================================
「Tools/Split」フォルダ以下

 1. 「in」フォルダに分割したい excel を配置します。複数あっても構いません。
 2. 「Split.bat」を実行します（※1）
 3. 「out」フォルダにシートの分だけ excel が生成されます。
 
　構成
 -------------
　フォルダ(分割したい　excel ファイル名) ---- シート1.xls
								  |-- シート2.xls
								  |-- ...
 ※1
 ・有給管理表に特化した形の分割ができます。
 ・詳しくは「SplitParam.xls」を参照。
 ・ 「MoveToPost.bat」実行すると、各ファイルを共有サーバーの各スタッフのフォルダに移動できます。 

3.結合について
==================================
「Tools/Merge」フォルダ以下

 1. 「in」フォルダに結合したい excel を配置します。複数あってもかいません（※2）
 2. 「Merge.bat」を実行します。
 3. 「out」フォルダに、全てのシートが結合された excel が生成されます。（※3）

 ※2
 ・配置構成は分割の時と同様です。

　構成
 -------------
　フォルダ(結合したい　excel ファイル名) ---- シート1.xls（シートの1番目しか結合されません）
								  |-- シート2.xls（〃）
								  |-- ...
 ※3
 ・有給管理表に特化した形で分割していた場合は、いい感じに結合してくれます。
